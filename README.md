# Mobilya Otomasyon Sistemi (C++)

Bu proje, C++ dili kullanılarak geliştirilmiş, küçük ölçekli bir mobilya mağazasının stok ve ürün kayıtlarını yönetmeyi amaçlayan bir konsol uygulamasıdır.

 Özellikler
* **Ürün Ekleme:** Yeni mobilya kayıtlarını ikili (binary) dosyalara kaydeder.
* **Stok Listeleme:** Kayıtlı tüm mobilyaları detaylıca listeler.
* **Ürün Arama:** ID numarasına göre hızlı arama yapar.
* **Ürün Silme:** Yedek dosya mantığı kullanarak kayıtları güvenli bir şekilde siler.
* **Ürün Düzenleme:** Mevcut ürün bilgilerini günceller.
* **Kullanıcı Dostu Menü:** Yanlış seçimlerde uyarı verir ve çerçeveli menü yapısına sahiptir.

 Dosya Yapısı
* `main.cpp`: Projenin tüm kaynak kodlarını içerir.
* `mobilya.dat`: Verilerin kalıcı olarak saklandığı ikili veri dosyası.

 Çalıştırma
Kodları derledikten sonra çıkan `.exe` dosyasını çalıştırarak otomasyonu kullanmaya başlayabilirsiniz.
